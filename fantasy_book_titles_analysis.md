# Fantasy Book Titles Analysis

This document contains the analysis of the first 10 fantasy book titles found on the Graphicaudio International website, cross-referenced with data found on Goodreads.

**Methodology:**
1. Titles were extracted from the Graphicaudio source page.
2. Goodreads data (Rating, Genres, Links) was sourced by performing specific search queries on the Goodreads platform for the most prominent book in each series.
3. The final table is sorted by Goodreads average rating in descending order.
